# Astro + Sanity blog starter

This code accompanies [a guide on getting started with Sanity and Astro](https://www.sanity.io/guides/sanity-astro-blog).

## Get started

1. Clone this repo
2. `npx sanity init --bare --env`
3. `npm install`
4. `npm run dev`
